# Lux's RPG
This is a game where you have to survive as long as you can.

Basic information:
- Life: Your health pool. If it reaches 0, you die. (Simple)
    - Importance: If health = 0, run function "die"
    - How to upgrade it:
        - Use the "Dragon Heart" item from the shop
        - Level up
- Mana: I wonder how you cast spells?
    - Importance
        - Use it for spells
    - How to upgrade it:
        - Use the "Eldritch Orb" item from the shop
        - Level up
- Sanity: Do you want to be insane or not?
    - Importance: If it reaches 0, you lose mana at the start of each day (Mana Lost scales with the amount of days survived)
    - How to upgrade it:
        - Use the "Eldritch Orb" item from the shop
        - Level up
- Mana Reduction: Wait what was this one again?
    - Importance:
        - Decreases the multiplier on spells when you increase your max mana
    - How to upgrade it:
        - Purchase the "Mana Stabilizer" from the shop
- Gold: Are we rich or are we not?
- EXP: Required to level up.
- LV: Gain a skill point when you level up
- SP: Skill points
- Spells: Deal damage or heal or increase your sanity. Better spells require more skill points.
